Technology research, API exploration, and best practices discovery with comparative analysis.

## Installation
```bash
/plugin install research
```

## Usage
```bash
/research "Compare Redux vs Zustand vs MobX for state management"
```

## Specialized Agents (5)
1. **information-gatherer**: Collects information about technologies
2. **feature-comparator**: Creates feature comparison matrices
3. **performance-evaluator**: Evaluates performance characteristics
4. **ecosystem-evaluator**: Assesses community health and resources
5. **recommendation-synthesizer**: Creates final recommendation

## Output
- RESEARCH_REPORT.md: Complete comparison and recommendation
- REFERENCES.md: Links and resources
